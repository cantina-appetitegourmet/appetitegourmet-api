# Appetite Gourmet: Rede de Cantinas Escolares
## Alimentação saudável e saborosa faz a diferença!

# Ferramenta para o backend

* Spring Boot 3.1.0
* Java 17
* Maven 3.9.2
* PostgreSQL 14
* Lombok
* Flyway
* JPA / Hibernate
* Bean Validation
* SpringDoc OpenAPI / Swagger UI

# Instalar dependências

`mvn install`

# Rodar projeto

`mvn spring-boot:run`
